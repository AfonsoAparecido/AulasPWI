Doctype Informa o navegador de uma possível versão que está sendo usada no Html
Html É a estrutura básica do html, Todo documento HTML deve se iniciar com essa tag
Head Ela delimita o cabeçalho do documento, é capaz de transmitir ao navegador informações mais úteis e essenciais
Title Tag que define o título da página, o nome que aparece na aba, janela ou guia, tem grande importãncia
Meta Essa tag permite inserir metadados como "charset=UTF-8", que garante compatibilidade com o código
Body Representa o corpo do documento, é nessa tag que todos os elementos devem ser inseridos na tag
